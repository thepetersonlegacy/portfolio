## üìä **Accessibility Contrast Ratios**

### **Text on White Background:**

| Color | Contrast Ratio | WCAG AA | WCAG AAA | Usage |
|-------|----------------|---------|----------|-------|
| **Red-600** (#D40000) | 7.2:1 | ‚úÖ Large Text | ‚úÖ Large Text | Headings, buttons only |
| **Red-900** (#7F1D1D) | 10.5:1 | ‚úÖ All Text | ‚úÖ All Text | Body text, small text |
| **Gold-500** (#EBC562) | 1.8:1 | ‚ùå Fails | ‚ùå Fails | **Backgrounds only** |
| **Gold-900** (#78350F) | 8.5:1 | ‚úÖ All Text | ‚úÖ All Text | Body text, small text |
| **Blue-600** (#0a2f85) | 9.8:1 | ‚úÖ All Text | ‚úÖ All Text | All text sizes |

### **Key Takeaways:**
- ‚úÖ **Use Red-600 for large text** (headings, buttons)
- ‚úÖ **Use Red-900 for small text** (body copy)
- ‚ùå **Never use Gold-500 for text** (use Gold-900 instead)
- ‚úÖ **Blue-600 is safe for all text sizes**
